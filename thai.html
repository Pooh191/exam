วิชาภาษาไทย รหัส ท ๒๓๑๐๑ (เวลา ๖๐ นาที)

สอบวันที่ ๓ ตุลาคม ๒๕๖๘ เวลา ๑๐.๑๐ – ๑๑.๑๐ น.

ข้อสอบปรนัย ๔๐ ข้อ : ๒๐ คะแนน

๑. ระดับภาษา (การเลือกใช้คำกับพระสงฆ์ + อ่านใบความรู้และเฉลยแบบฝึกหัด)
เทคนิค : อ่านข้อความแล้วจับผิดว่ามีคำภาษาพูดปนอยู่หรือไม่ ถ้ามีจะเป็น "กึ่งทางการ" ทันที
ตัวอย่างข้อสอบระดับภาษา

ข้อ 1. ข้อใดใช้ภาษาได้เหมาะสมกับสถานการณ์

ก. หลวงพ่อครับ ผมขออนุญาตถวายปัจจัยเพื่อสมทบทุนสร้างพระอุโบสถ

ข. ท่านพระอาจารย์ ขอให้ช่วยเทศน์สอนวิธีทำสมาธิให้หน่อยครับ

ค. เณรน้อย กรุณานำทางไปยังกุฏิพระอาจารย์ด้วย

ง. สวัสดีครับหลวงพี่ วันนี้ดูหน้าตาแจ่มใสจัง

คำตอบ: ข้อ ก
เหตุผล: ใช้คำว่า "ถวาย" ซึ่งเหมาะสมกับการใช้กับพระสงฆ์ และใช้ภาษาทางการ

ตารางสรุประดับภาษา ๕ ระดับ

ระดับภาษา ลักษณะภาษา ข้อสังเกต/พบบ่อย พบใน
พิธีการ เขียน เป็นพิธีรีตอง ใช้ภาษาสูง ภาษาไพเราะ สละสลวย การเปิดประชุม / การกล่าวรายงาน การกล่าวสุนทรพจน์ / การกล่าวปราศรัย การกล่าวสดุดี / การกล่าวต้อนรับ การกล่าวอวยพร
ทางการ (ออกสอบมากที่สุด) เขียน ภาษาทางการ ถ้อยคำตรงไปตรงมา มีศัพท์เทคนิคหรือศัพท์วิชาการบ้าง (ชัดเจน + กระชับ) งานสัมมนาวิชาการ (คนมาก ๆ) รายงานเชิงวิชาการ / หนังสือราชการ จดหมายราชการ / คำนำหนังสือ ประกาศของทางราชการ
กึ่งทางการ พูด + เขียน (ขึ้นกับบริบท) ใช้ภาษาเขียน แต่มีภาษาพูดแทรกอยู่บ้าง ภาษาคุ้นเคยกว่าทางการ ใช้ศัพท์เฉพาะเท่าที่จำเป็น การประชุมกลุ่มย่อย / การอภิปรายกลุ่ม / ข่าว การเสวนา / การบรรยายในห้องเรียน (ครูสอน) รายงานหน้าชั้นเรียน / บทความในหนังสือพิมพ์
ไม่เป็นทางการ (สนทนา) พูด ใช้ภาษาพูดที่สุภาพ การสนทนาโต้ตอบไม่เกิน ๕ คน อยู่ในสถานที่และเวลาที่ไม่เป็นส่วนตัว (เน้นความสัมพันธ์ระหว่างกัน คำทำไป แต่ไม่หยาบ) จดหมายส่วนตัว / ภาษาในบล็อกเกอร์ ภาษาในหนังสือพิมพ์ (พาดหัวข่าว) บทความแสดงความคิดเห็นในนิตยสารหรือหนังสือพิมพ์ รายการสาระบันเทิงทางโทรทัศน์ การปรึกษาหารือกัน / ประชุมกลุ่มย่อย (เน้นสนิท)
กันเอง พูด ใช้ภาษาพูด อาจมีคำคะนอง(สแลง) หรือภาษาอื่นต่าง ๆ การสนทนาในชีวิตประจำวัน การสนทนาเรื่องส่วนตัว การทักทายระหว่างเพื่อนสนิท
๒. ย่อความ (หาใจความสำคัญ = มักอยู่ช่วงต้นของบทความ)
ตัวอย่างข้อสอบย่อความ

ข้อ 2. ให้นักเรียนอ่านบทความต่อไปนี้แล้วตอบคำถาม

"การอ่านหนังสือเป็นกิจกรรมที่มีประโยชน์อย่างยิ่งต่อการพัฒนาสติปัญญาและความคิดของมนุษย์ หนังสือช่วยเปิดโลกทัศน์ให้กว้างไกล ทำให้เราได้เรียนรู้ประสบการณ์ของผู้อื่น และยังช่วยพัฒนาทักษะทางภาษาทั้งการฟัง พูด อ่าน และเขียน ผู้ที่รักการอ่านมักจะเป็นผู้ที่มีความรู้กว้างขวางและมีเหตุผล"

ใจความสำคัญของบทความนี้คืออะไร?

คำตอบ: การอ่านหนังสือมีประโยชน์ต่อการพัฒนาสติปัญญาและความคิดของมนุษย์
๓. จดหมาย (ระดับภาษา = ภาษาทางการ / คำขึ้นต้น+ลงท้าย = เรียน – ขอแสดงความนับถือ)
ตัวอย่างข้อสอบจดหมาย

ข้อ 3. ข้อใดเป็นคำลงท้ายจดหมายราชการที่ถูกต้อง

ก. ด้วยความปราถนาดี

ข. ขอแสดงความนับถือ

ค. รักและคิดถึงเสมอ

ง. ด้วยความเคารพอย่างสูง

คำตอบ: ข้อ ข
เหตุผล: "ขอแสดงความนับถือ" เป็นคำลงท้ายจดหมายราชการที่ถูกต้องตามแบบแผน
๔. แต่งบทร้อยกรอง (กลอนสุภาพ / กาพย์ยานี ๑๑ / โคลงสี่สุภาพ : อ่านใบความรู้และแบบฝึกหัดที่ครูให้เดิม)
  • คำเอก = ใช้ "คำตาย" แทนได้
  • คำเอกโทษ = มองเห็นเป็น "รูปเอก" แต่ในอดีตเคยเป็นรูปอื่นมาก่อน เช่น หน้า เปลี่ยนเป็น น่า
  • คำโทโทษ = มองเห็นเป็น "รูปโท" แต่ในอดีตเคยเป็นรูปอื่นมาก่อน เช่น ที่ เปลี่ยนเป็น ลี้
ตัวอย่างข้อสอบแต่งบทร้อยกรอง

ข้อ 4. ให้นักเรียนเติมคำในช่องว่างของโคลงสี่สุภาพต่อไปนี้ให้สัมผัสถูกต้อง

เหมือนชีวิตเราหมดดาย"

ข้อ 5. จากการสำรวจพบว่า นักเรียนไทยใช้เวลาอ่านหนังสือเฉลี่ย ๒ ชั่วโมงต่อวัน ในขณะที่นักเรียนในประเทศฟินแลนด์ใช้เวลาอ่านหนังสือเฉลี่ย ๔ ชั่วโมงต่อวัน และนักเรียนในประเทศญี่ปุ่นใช้เวลาอ่านหนังสือเฉลี่ย ๓ ชั่วโมงต่อวัน

จากข้อมูลข้างต้น ให้นักเรียนวิเคราะห์ว่าปัจจัยใดอาจส่งผลต่อพฤติกรรมการอ่านของนักเรียนไทย

คำตอบตัวอย่าง: ปัจจัยที่อาจส่งผลได้แก่ วัฒนธรรมการอ่านที่ยังไม่เข้มแข็ง การแข่งขันจากสื่อดิจิทัลและโซเชียลมีเดีย ระบบการศึกษาที่เน้นการท่องจำมากกว่าการส่งเสริมการอ่านอย่างต่อเนื่อง และสภาพแวดล้อมในบ้านที่อาจไม่เอื้อต่อการอ่านหนังสือ
